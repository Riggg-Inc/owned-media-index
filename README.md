# Owned Asset Media Index

The **Owned Asset Media Index** is a living, community-maintained repository documenting best practices for creating and operating episodic owned media assets, starting with podcasts.

This is a decision library for podcasters, operators, marketers, creators, and production teams who want to move fast with proven patterns.

Owned by **Riggg Inc.**

## Purpose

The index exists to answer one practical question:

**What does good look like when creating, publishing, and operating an episodic owned media asset?**

Every entry should answer:

1. **What is this for?** Use case and context.
2. **When do you use it?** Trigger conditions, skill level, and budget tier.
3. **What does good look like?** Quality bar, benchmark, or minimum standard.

## Sections

### Strategy

Editorial and business foundation before any tool or gear is touched.

Topics include audience mapping, show concept design, format selection, and how a podcast connects to measurable business outcomes such as lead generation, SEO, authority, sales enablement, and trust building.

### Stack

Software organized by functional category:

- Recording
- Editing
- Cleanup and mastering
- Hosting
- Transcription
- Repurposing
- Collaboration
- Analytics

Each tool entry should include purpose, pricing tier, best-fit scenarios, alternatives, and a Riggg score from 1 to 5.

### Hardware

Gear recommendations organized by tier:

- Starter: under $500
- Mid-tier: $500-$2,000
- Pro: $2,000+

Hardware entries should cover microphones, headphones, interfaces/recorders, cameras, lighting, switching/capture, and room treatment.

### Content Formats

A registry of repeatable episode formats:

- Expert Interview
- Operator Roundtable
- Solo Explainer
- Case Study Breakdown
- Tactical Briefing

Each format includes use cases, structure, planning templates, production considerations, and repurposing potential.

### Publishing and Distribution

The release system for owned media.

The index is website-first: the owned website is the canonical home for each episode. Distribution then extends to podcast platforms, YouTube, social clips, email, guest-share assets, and conversion tracking.

## Community Model

The index uses a two-tier contribution model:

- **Community Contributor:** anyone can open issues, suggest tools, submit pull requests, and share production notes.
- **Riggg-Reviewed:** entries reviewed and accepted by Riggg against the scoring criteria.

Merged entries may be tagged as `riggg-reviewed` after review.

## Scoring

Tools and workflows are scored from 1 to 5:

- **5:** Industry standard. Recommended default.
- **4:** Strong fit. Used regularly in Riggg workflows.
- **3:** Viable option. Works with tradeoffs noted.
- **2:** Niche case. Specific use case only.
- **1:** Not recommended. Included for completeness or context.
