# Roadmap

The Owned Media Index should grow from a useful pattern library into a quarterly authority publication.

## V0.1 - Framework

Status: in progress

- Define Produce -> Package -> Publish -> Prove -> Preserve.
- Create repository structure.
- Add pattern schema.
- Add scoring and evidence rules.
- Seed initial patterns.

## V0.2 - Packaging Patterns

Focus:

- Title styles.
- RSS descriptions.
- YouTube descriptions.
- LinkedIn posts.
- Clip selection.
- Reel packaging.
- Thumbnail hooks.
- Chapters.
- Guest-share assets.

Goal:

Create the first practical library of repeatable packaging patterns.

## V0.3 - Publishing Patterns

Focus:

- Website-first publishing.
- RSS feed standards.
- YouTube release standards.
- LinkedIn distribution.
- Email release patterns.
- Live premieres.
- Simulcast distribution.
- Distribution calendar structure.

Goal:

Make distribution a first-class part of the owned media system.

## V0.4 - Tools and Hardware

Focus:

- Recording tools.
- Editing tools.
- Mastering tools.
- Transcription tools.
- Repurposing tools.
- Publishing tools.
- Analytics tools.
- Hardware by use case.

Goal:

Create ranked decision support for teams building their owned media stack.

## V0.5 - Prove and Preserve

Focus:

- Scorecards.
- Benchmarks.
- Conversion events.
- Attribution patterns.
- Transcript standards.
- Vector memory.
- Searchability.
- Content reuse.

Goal:

Document how owned media becomes measurable and compounding.

## V1.0 - First Quarterly Issue

Focus:

The first public issue of the Owned Media Index.

Recommended theme:

**Distribution**

Why:

Distribution connects directly to Riggg's strongest market claim: owned media should not stop at asset creation. It should be produced, packaged, published, measured, and preserved.
