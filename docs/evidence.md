# Evidence Rules

The Owned Media Index should be opinionated, but not hand-wavy.

Each entry should explain what kind of evidence supports it.

## Evidence Types

- **Internal production data:** private learning generalized into public patterns.
- **Teardown observation:** analysis of public examples.
- **Platform documentation:** official requirements and guidance.
- **Practitioner observation:** expert operating judgment.
- **External research:** credible reports, studies, or public datasets.

## Public Safety

Do not publish confidential customer information, private Airtable data, private Slack messages, unapproved performance metrics, or internal workflow secrets.

When using internal learning, generalize it into a public pattern.
