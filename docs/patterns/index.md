# Pattern Library

Patterns are reusable answers to production, packaging, publishing, measurement, and preservation problems.

Every pattern should explain:

- What it is.
- What it is best for.
- Why it works.
- What good looks like.
- When not to use it.
- What evidence supports it.

## How To Read Scores

Scores range from 1 to 5.

- **5:** Recommended default.
- **4:** Strong fit.
- **3:** Viable with tradeoffs.
- **2:** Niche.
- **1:** Not recommended as a default.
