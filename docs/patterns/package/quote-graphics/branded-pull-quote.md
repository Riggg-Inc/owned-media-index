# Branded Pull Quote

**Stage:** Package → Quote Graphics  
**Score:** 4  
**Evidence:** practitioner observation

## What It Is

A templatized quote graphic using consistent brand colors, fonts, and layout across every episode. Builds show recognition in feeds.

## Quality Bar

- Quote must be 30 words or fewer
- Speaker attribution: name, title, company
- Readable at mobile size without zooming
- Brand colors and fonts consistent
- Works at 1:1 and 16:9 aspect ratios

## When Not To Use

Avoid when template fatigue is reducing engagement — rotate designs periodically.

## Prompt Template

Copy and customize this prompt to generate this pattern:

```
Create a quote graphic using the Branded Pull Quote pattern.

Context:
- Quote or content: [the text to feature]
- Speaker(s): [name, title, company]
- Show brand colors: [hex codes]
- Episode reference: [episode title or number]

Requirements:
- Text must be 30 words or fewer
- Speaker attribution clear
- Readable at mobile size
- Both 1:1 and 16:9 versions
- Brand consistent
```

Replace the bracketed placeholders with your specific details. The more context you provide about your audience, guest, and episode content, the better the output.
