# Branded Pull Quote

**Stage:** Package → Quote Graphics  
**Score:** 4  
**Evidence:** practitioner observation

## What It Is

A templatized quote graphic using consistent brand colors, fonts, and layout across every episode. Builds show recognition in feeds.

## Quality Bar

- Quote must be 30 words or fewer
- Speaker attribution: name, title, company
- Readable at mobile size without zooming
- Brand colors and fonts consistent
- Works at 1:1 and 16:9 aspect ratios

## When Not To Use

Avoid when template fatigue is reducing engagement — rotate designs periodically.
