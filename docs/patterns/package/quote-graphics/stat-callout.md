# Stat Callout

**Stage:** Package → Quote Graphics  
**Score:** 4  
**Evidence:** practitioner observation

## What It Is

A graphic that highlights a specific data point or benchmark from the session, with the number as the dominant visual element and context text below.

## Quality Bar

- Quote must be 30 words or fewer
- Speaker attribution: name, title, company
- Readable at mobile size without zooming
- Brand colors and fonts consistent
- Works at 1:1 and 16:9 aspect ratios

## When Not To Use

Avoid when the statistic requires heavy context or when the source is not credible.
