# Hook Insight Cta

**Stage:** Package → Reels  
**Score:** 5  
**Evidence:** platform observation

## What It Is

A reel that opens with a visual or verbal hook, delivers one tactical insight, and closes with a clear CTA. The most reliable reel format for owned media.

## Why It Works

The three-part structure mirrors how short-form platforms reward content: stop the scroll (hook), deliver value (insight), and convert attention (CTA). Each section has a job.

## Quality Bar

- Hook must land in the first 2 seconds
- Captions must be burned in and readable on mobile
- Vertical native (9:16) — never a cropped horizontal video
- One clear idea per reel (except rapid-fire)
- CTA must be specific: follow, subscribe, link in bio, full episode

## When Not To Use

Avoid when the insight requires more than 30 seconds of context to understand.

## Prompt Template

Copy and customize this prompt to generate this pattern:

```
Write a reel script using the Hook-Insight-CTA pattern.

Context:
- Episode topic: [what is this episode about]
- Key insight: [the one tactical takeaway]
- Target audience: [who is this for]
- CTA: [follow, subscribe, link in bio, full episode]

Format:
HOOK (0-2s): [visual/verbal hook that stops the scroll]
INSIGHT (2-25s): [deliver the one tactical insight]
CTA (25-30s): [clear call to action]

Requirements:
- Hook must land in first 2 seconds
- One clear idea only
- Include caption text (80%+ watch without sound)
- 15-30 seconds total
```

Replace the bracketed placeholders with your specific details. The more context you provide about your audience, guest, and episode content, the better the output.
