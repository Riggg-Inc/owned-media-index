# Rapid Fire

**Stage:** Package → Reels  
**Score:** 4  
**Evidence:** platform observation

## What It Is

A reel that delivers 3-5 quick insights, tips, or takes in rapid succession. Each point gets 5-10 seconds. Pacing keeps attention through the full reel.

## Why It Works

Multiple quick hits keep the viewer watching. Each new point resets the attention clock. The format also maximizes information density per second.

## Quality Bar

- Hook must land in the first 2 seconds
- Captions must be burned in and readable on mobile
- Vertical native (9:16) — never a cropped horizontal video
- One clear idea per reel (except rapid-fire)
- CTA must be specific: follow, subscribe, link in bio, full episode

## When Not To Use

Avoid when the insights are too nuanced for 5-second treatment.

## Prompt Template

Copy and customize this prompt to generate this pattern:

```
Write a Rapid Fire reel script with 3-5 quick insights.

Context:
- Episode topic: [what is this about]
- Insights (3-5): [list each quick insight]
- Target audience: [who is this for]

Format:
HOOK (0-2s): ["3 things every [audience] should know about [topic]"]
POINT 1 (2-8s): [insight + text overlay]
POINT 2 (8-14s): [insight + text overlay]
POINT 3 (14-20s): [insight + text overlay]
CTA (20-25s): [call to action]

Requirements:
- Each point gets 5-8 seconds max
- Text overlays for each point
- Pacing must be energetic
- 20-30 seconds total
```

Replace the bracketed placeholders with your specific details. The more context you provide about your audience, guest, and episode content, the better the output.
