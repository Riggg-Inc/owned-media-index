# Teaser Trailer

**Stage:** Package → Reels  
**Score:** 3  
**Evidence:** platform observation

## What It Is

A reel designed to promote an upcoming or recent episode. Combines the best 2-3 moments with text overlays and a release date or link.

## Why It Works

Trailers build anticipation and give the audience a reason to come back. They also perform well as paid promotion content.

## Quality Bar

- Hook must land in the first 2 seconds
- Captions must be burned in and readable on mobile
- Vertical native (9:16) — never a cropped horizontal video
- One clear idea per reel (except rapid-fire)
- CTA must be specific: follow, subscribe, link in bio, full episode

## When Not To Use

Avoid overusing — one trailer per episode is enough. More feels like advertising.

## Prompt Template

Copy and customize this prompt to generate this pattern:

```
Write a Teaser Trailer reel script to promote an episode.

Context:
- Episode title: [title]
- Guest name: [who is featured]
- Best 2-3 moments: [brief description of highlight clips]
- Release date: [when does it drop]
- Where to listen/watch: [platform links]

Format:
TEASE 1 (0-5s): [best moment clip or quote]
TEASE 2 (5-10s): [second highlight]
TITLE CARD (10-15s): [episode title + guest + release date]
CTA (15-20s): [where to watch/listen]

Requirements:
- Best moments first — no slow buildup
- Title card must be readable
- Include release date or "out now"
- 15-20 seconds
```

Replace the bracketed placeholders with your specific details. The more context you provide about your audience, guest, and episode content, the better the output.
