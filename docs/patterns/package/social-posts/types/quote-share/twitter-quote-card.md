# Twitter Quote Card

**Stage:** Package → Social Posts → Quote Share  
**Score:** 4  
**Evidence:** practitioner observation

## Platform

Twitter/X

## What It Is

A platform-specific version of the Quote Share social post type, optimized for Twitter/X's short-form, engagement-driven format.

## Template

```
[Hook — first line earns the second line]

[Body — value, context, or story]

[CTA — one clear action]

[Tags/Hashtags if appropriate]
```

## Quality Bar

- First line must stop the scroll
- Body must add value beyond the episode title
- CTA must be specific and singular
- Platform-native formatting (line breaks, emojis, hashtags per platform norms)
