# Thumbnail Hook Patterns

**Stage:** Package  
**Score:** 4  
**Evidence:** practitioner observation

## What It Is

Reusable patterns for thumbnail concepts that help an owned media asset earn attention without misleading the audience.

## Example Patterns

1. The Tension
2. The Expert
3. The Outcome
4. The Mistake
5. The Comparison
6. The Framework
7. The Question
8. The Moment
9. The Object
10. The Event

## Quality Bar

The thumbnail should communicate the asset's point at small size.

## When Not To Use

Avoid overly busy thumbnails, misleading emotion, tiny text, and generic stock imagery.
