# Data Visual

**Stage:** Package → Thumbnails  
**Score:** 3  
**Evidence:** platform observation

## What It Is

A thumbnail featuring a key statistic, chart element, or number as the primary visual. Works when the data point is the hook.

## Quality Bar

- Readable at 120x120px minimum
- Maximum 5-7 words of text
- High contrast between text and background
- Brand colors consistent with show identity
- Face visible when using headshot or emotion styles

## When Not To Use

Avoid when the data point requires context to understand or when the number is not surprising.
