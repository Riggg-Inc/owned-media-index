# Scene Still

**Stage:** Package → Thumbnails  
**Score:** 3  
**Evidence:** platform observation

## What It Is

A thumbnail using a production still from the actual recording session. Shows the studio, setup, or behind-the-scenes moment.

## Quality Bar

- Readable at 120x120px minimum
- Maximum 5-7 words of text
- High contrast between text and background
- Brand colors consistent with show identity
- Face visible when using headshot or emotion styles

## When Not To Use

Avoid when the recording setup is not visually interesting or professional.
