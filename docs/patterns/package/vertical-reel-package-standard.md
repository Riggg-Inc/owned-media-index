# Vertical Reel Package Standard

**Stage:** Package  
**Score:** 5  
**Evidence:** practitioner observation

## What It Is

A standard for packaging a short-form vertical video asset from a longer session.

## Required Elements

- 9:16 aspect ratio.
- Clear speaker framing.
- Captions.
- Hook or title where appropriate.
- Clean audio.
- Platform-safe margins.
- Description/caption copy.
- Source session link or CTA when appropriate.

## Quality Bar

The reel should be understandable with sound off and valuable with sound on.

## When Not To Use

Avoid vertical reels when the source video relies on wide visuals, dense screen shares, or context that cannot survive a short-form crop.
