# Canonical Episode Page

**Stage:** Publish  
**Score:** 5  
**Evidence:** practitioner observation

## What It Is

The owned website page that acts as the canonical home for an episode, webinar, livestream replay, or virtual event session.

## Required Elements

- Embedded audio and/or video player.
- Episode/session title.
- Summary.
- Key takeaways.
- Timestamped transcript.
- Chapters.
- Guest bio and links.
- Related resources.
- Clear CTA.
- Schema markup where appropriate.

## Quality Bar

The page should be useful even if the visitor never presses play.

## When Not To Use

Do not create thin episode pages that only embed a player and duplicate an RSS blurb.
