# RSS Feed Metadata Standard

**Stage:** Publish  
**Score:** 5  
**Evidence:** practitioner observation

## What It Is

A standard for podcast RSS metadata that supports discoverability, clarity, and distribution quality.

## Required Fields

- Episode title.
- Episode description.
- Publish date.
- Audio file.
- Duration.
- Explicit status.
- Episode artwork where applicable.
- Episode number or season where applicable.
- Guest name where applicable.
- Canonical website link.

## Quality Bar

The metadata should make the episode understandable inside a podcast app without requiring the listener to visit another page.
