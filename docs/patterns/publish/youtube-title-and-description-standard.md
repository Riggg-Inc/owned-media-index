# YouTube Title and Description Standard

**Stage:** Publish  
**Score:** 5  
**Evidence:** practitioner observation

## What It Is

A standard for packaging owned media assets for YouTube.

## Required Elements

- Clear title.
- Description with episode/session summary.
- Guest names and roles.
- Chapters where possible.
- Links to canonical page and resources.
- CTA.
- Relevant tags where appropriate.
- Thumbnail aligned with title.

## Quality Bar

The YouTube package should help a viewer decide what the video is about, navigate to relevant sections, and take the next step.

## When Not To Use

Avoid stuffing descriptions with unrelated keywords or treating YouTube as a file dump for unprocessed recordings.
