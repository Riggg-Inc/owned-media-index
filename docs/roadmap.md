# Roadmap

The Owned Media Index should grow from a useful pattern library into a quarterly authority publication.

## V0.1 - Framework

- Define Produce -> Package -> Publish -> Prove -> Preserve.
- Create repository structure.
- Add pattern schema.
- Add scoring and evidence rules.
- Seed initial patterns.

## V0.2 - Packaging Patterns

Title styles, RSS descriptions, YouTube descriptions, LinkedIn posts, clip selection, reel packaging, thumbnail hooks, chapters, and guest-share assets.

## V0.3 - Publishing Patterns

Website-first publishing, RSS feed standards, YouTube release standards, LinkedIn distribution, email release patterns, live premieres, simulcast distribution, and distribution calendar structure.

## V0.4 - Tools and Hardware

Ranked decision support for recording, editing, mastering, transcription, repurposing, publishing, analytics, and hardware.

## V1.0 - First Quarterly Issue

Recommended first theme: **Distribution**.

Distribution connects directly to the core claim: owned media should not stop at asset creation.
