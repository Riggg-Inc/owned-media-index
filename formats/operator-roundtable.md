# Operator Roundtable

Multi-guest discussion targeting one problem with multiple operators.

## What This Is For

Operator roundtables capture peer-level B2B insight and differentiated discussion.

## When To Use It

Use this format when:

- The topic benefits from multiple perspectives.
- Guests are credible operators.
- The host can moderate tightly.
- The goal is category or market insight.

## What Good Looks Like

Strong roundtables include:

- Theme selection.
- Guest selection matrix.
- Discussion arc planning.
- Clear moderation rules.
- Tight time control.
