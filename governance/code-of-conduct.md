# Code of Conduct

This project is for practical, good-faith documentation of episodic owned media best practices.

## Expected Behavior

- Be specific and constructive.
- Share evidence where possible.
- Disclose affiliation when recommending tools.
- Respect contributors.
- Focus on production usefulness.

## Unacceptable Behavior

- Harassment.
- Spam.
- Undisclosed promotional submissions.
- Personal attacks.
- Low-effort tool dumping.

## Enforcement

Riggg may close issues, reject pull requests, remove comments, or block contributors who do not follow this code of conduct.
