# Contribution Guide

The Owned Asset Media Index uses a two-tier contribution model.

## Community Contributor

Community contributors can:

- Open issues suggesting tools, formats, or corrections.
- File pull requests with new entries or updates.
- Share production experience in GitHub Discussions.

## Riggg-Reviewed

Riggg-reviewed entries have been evaluated against the scoring criteria and accepted by the Riggg editorial team.

## Review Flow

1. Community member opens an issue or pull request.
2. Riggg reviews against scoring criteria.
3. Accepted entries are merged and tagged.
4. Rejected entries receive feedback.
5. Quarterly review flags stale or outdated entries.
