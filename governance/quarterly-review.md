# Quarterly Review

The index should be reviewed quarterly to keep recommendations current.

## Review Checklist

- Check stale tool entries.
- Update pricing.
- Review discontinued products.
- Review new platform requirements.
- Re-score tools where needed.
- Flag outdated workflows.
- Add changelog entry.

## Stale Entry Criteria

An entry may be stale if:

- Pricing changed materially.
- Core features changed.
- Product quality declined.
- Better alternatives emerged.
- Integration or export behavior changed.
- The entry has not been reviewed in more than one year.
