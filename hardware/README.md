# Hardware

Gear recommendations for episodic owned media production.

The goal is not to recommend the most expensive setup. The goal is to help teams choose reliable gear that fits their use case, budget, and production environment.

## Tiers

- **Starter:** budget setup under $500.
- **Mid-tier:** prosumer setup from $500-$2,000.
- **Pro:** studio-grade setup above $2,000.

## Evaluation Criteria

Hardware should be scored on:

- Reliability.
- Value.
- Ease of setup.
- Audio/video quality.
- Fit for remote, in-studio, or mobile use.
- Upgrade path.

## Categories

- Microphones.
- Headphones.
- Interfaces and recorders.
- Cameras.
- Lighting.
- Switching and capture.
- Room treatment.
