# Distribution Checklist

Checklist for releasing each episode.

## Release Steps

1. Publish to the owned website first.
2. Submit to Apple Podcasts, Spotify, Amazon Music, and Pocket Casts via RSS.
3. Upload video version to YouTube with chapters.
4. Generate and post 3 to 5 social clips.
5. Send episode email with embedded player and CTA.
6. Create guest-share package.
7. Log release in analytics dashboard.
8. Set conversion events.
