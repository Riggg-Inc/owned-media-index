# Owned Channel Standard

The owned website is the canonical home for each episode.

## What This Is For

This standard defines the minimum episode page requirements for website-first publishing.

## When To Use It

Use this for every published episode.

## What Good Looks Like

Every episode should have a dedicated page on the owned website with:

- Embedded player for audio and/or video.
- Full timestamped transcript.
- Episode summary with key takeaways.
- Timestamps and chapter markers.
- Guest bio and resource links.
- Clear CTA.
- Schema.org `PodcastEpisode` markup where applicable.

## Why Website First

Podcast apps and social platforms are distribution channels. The owned website is the durable home.

Publishing to the owned website first creates:

- Search value.
- Conversion paths.
- Ownership.
- Better attribution.
- A canonical link for guest and social sharing.
