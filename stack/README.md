# Stack

Software categories for producing and operating episodic owned media.

Each tool entry should answer:

1. What is this for?
2. When do you use it?
3. What does good look like?

## Categories

| Category | Purpose | Quality Bar |
| --- | --- | --- |
| Recording | Capture audio/video of hosts and guests | Local backup recording required |
| Editing | Assembly, cuts, pacing, transitions | Consistent loudness target, usually -16 LUFS for stereo podcast audio |
| Cleanup and Mastering | Noise reduction, leveling, final output | Consistent low noise floor and intelligible speech |
| Hosting | RSS feed, storage, analytics | Episode pages on own domain required |
| Transcription | Auto-generate text and captions | Timestamps and searchability required |
| Repurposing | Turn episodes into clips, summaries, posts | One episode produces at least 5 derivative assets |
| Collaboration | Guest booking, file sharing, async workflow | Backup recording workflow required |
| Analytics | Measure reach, engagement, conversions | Per-episode conversion events tracked |
